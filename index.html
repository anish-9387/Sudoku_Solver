How It Works

  1. Find an empty cell.
  2. Try numbers 1-9 in that cell.
  3. Check if the number is safe (no conflicts in row, column, or 3x3 box).
  4. If safe → place the number and recurse to the next empty cell.
  5. If no number works → backtrack (erase & try the next option).
  6. Repeat until the board is solved or proven unsolvable.
