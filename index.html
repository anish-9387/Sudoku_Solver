How Backtracking Works

  1. Find an empty cell in the grid.
  2. Try digits 1–9 in that position.
  3. Validate against row, column & 3×3 box.
  4. If valid → place & recurse deeper.
  5. If no digit works → backtrack to last choice.
  6. Repeat until solved or proven unsolvable.
